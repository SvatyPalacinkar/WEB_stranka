Čo je Python

  • Python is a popular programming language. It was created by Guido van Rossum, and released in 1991.
  • It is used for:
    • web development (server-side)
    • software development
    • mathematics
    • system scripting.

What can Python do?

  • Python can be used on a server to create web applications.
  • Python can be used alongside software to create workflows.
  • Python can connect to database systems. It can also read and modify files.
  • Python can be used to handle big data and perform complex mathematics.
  • Python can be used for rapid prototyping, or for production-ready software development.

Premenné v Pythone

  • Premenné slúžia na uloženie hodnoty

Vytváranie premnenných

  • V Pythone nieje príkaz na vytvorenie premennej
  • Premenná sa vytvorí v moment keď jej priradíme hodnotu

Priklad

x = 5
y = "Marek"


  • Premenné nepotrebujú mať deklarovaný dátový typ, a ten sa môže aj zmeniť po ich vytvorení

priklad

x = 4       #<= je dátový typ celé číslo
x = "Ahoj"  #<= je dátový typ reťazec


Pretypovanie

  • Ak je to potrebné tak sa špecifikovať dátový typ premnennej, volá sa to typovanie a pretypovanie

priklad

x = str(8)   #<= je '8'
x = int(8)   #<= je 8
x = float(8) #<= je 8.0
