Čo je Python

  • Python je populárny programovací jazyk. Bol vytvorený Guidom van Rossumom, a bol publikovaný v 1991.
  • Používa sa na:
    • Vývoj webstránok
    • Softwareový vývoj
    • Matematiku
    • Skyptovanie systému

Čo Python dokáže?

  • Python môže byť využitý na servery na vytváranie webových aj systemových aplikácií.
  • Python môže byť využívaný na spravovanie databázových systémov. Môže taktiež čítať a upravovať súbory.
  • Python môže spravovať veľké dáta a vykonávať komplexnú matematiku.

Priklad

print("Hello world!")

Premenné v Pythone

  • Premenné slúžia na uloženie hodnoty

Vytváranie premnenných

  • V Pythone nieje príkaz na vytvorenie premennej
  • Premenná sa vytvorí v moment keď jej priradíme hodnotu

Priklad

x = 5
y = "Marek"


  • Premenné nepotrebujú mať deklarovaný dátový typ, a ten sa môže aj zmeniť po ich vytvorení

priklad

x = 4       #<= je dátový typ celé číslo
x = "Ahoj"  #<= je dátový typ reťazec


Pretypovanie

  • Ak je to potrebné tak sa špecifikovať dátový typ premnennej, volá sa to typovanie a pretypovanie

priklad

x = str(8)   #<= je '8'
x = int(8)   #<= je 8
x = float(8) #<= je 8.0


Citlivosť znakov

  • Mená premenných rozoznávajú rozdiel medzi malími a veľkými písmenkami

priklad

a = 55
A = "text"
#A neprepíše a







Operátory v Pythone

  • Python delí Operátory do týchto skupín:
    • Aritmetické operátory
    • Priraďovacie operátory
    • Porovnávacie operátory

Aritmetické operátory
Operator Meno Priklad Vysledok
+ Súčet 10 + 5 15
- Rozdiel 50 - 5 45
* Súčin 5 * 5 25
/ Podiel 20 / 4 5
% Modulo 24 / 5 4
** Mocnina 2 ** 3 8
// Zaokrúhlené delenie 15 // 2 7


Priraďovacie operátory
Operator Priklad Rovnaké ako
+= x += 5 x = x + 5
-= x -= 5 x = x - 5
*= x *= 5 x = x * 5
/= x /= 5 x = x / 5
%= x %= 5 x = x % 5
//= x //= 5 x = x //5
**= x **= 5 x = x ** 5


Porovnávacie operátory
Operator Meno Príklad
== Je rovné x == y
!= Nieje rovné x != y
> Je väčšie ako x > y
< Je menšie ako x < y
>= Je väčšie alebo rovné x >= y
<= Je menšie alebo rovné x <= y
